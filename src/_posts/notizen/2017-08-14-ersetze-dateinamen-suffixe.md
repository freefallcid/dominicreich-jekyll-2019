Hier notiert weil ich mir die Syntax von `basename` noch nie merken konnte...

``` bash
for i in *.eml; do mv "$i" $(basename -s .eml "$i").txt; done
```

Obiger Einzeiler nennt alle `eml`-Dateien in `txt`-Dateien um.
